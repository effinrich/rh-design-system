# VisuallyHidden

VisuallyHidden is a common technique used in web accessibility to hide content
from the visual client, but keep it readable for screen readers.

## Import

```js
import { VisuallyHidden } from '@chakra-ui/react'
```

## Usage

<Playground
  scope={{ Button, VisuallyHidden, CheckIcon }}
  code={`
function Example() {
  return (
    <Button>
      <VisuallyHidden>Checkmark</VisuallyHidden>
      <CheckIcon />
    </Button>
  )
}
`}
/>

}

/>

---

## Accessibility

VisuallyHidden has all the styles necessary to hide it from visual clients, but
keep it for screen readers.
